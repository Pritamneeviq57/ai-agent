# Teams Meeting Summarization System

## Overview
Automated Teams meeting summarization using Microsoft Graph API and local LLMs (Ollama).

## Setup
1. Clone the repository
2. Create `.env` file from `.env.example`
3. Install dependencies: `pip install -r requirements.txt`
4. Run: `python main_phase_2_3_delegated.py`

## Requirements
- Python 3.8+
- Ollama (for local LLM)
- Microsoft Teams Graph API access
